# Real Estate Project

This project contains a Streamlit dashboard for analyzing and visualizing real estate data.

## Features
- Explore house sales data
- View interactive maps and charts
- Use a trained model for predictions

## Setup
1. Create a virtual environment
   python -m venv venv
   .\venv\Scripts\activate
2. Install dependencies
   pip install -r requirements.txt
3. Run the dashboard
   streamlit run dashboard/app.py

## Notes
- The dataset is stored in `data/kc_house_data.csv`.
- Trained model artifacts are stored in `models/`.
