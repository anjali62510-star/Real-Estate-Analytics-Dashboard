# 🏠 Real Estate Analytics Dashboard

An interactive Real Estate Analytics Dashboard built using **Python, Streamlit, Machine Learning, and Data Visualization**. The application enables users to explore housing market trends, analyze property data, and predict house prices using a trained machine learning model.

---

## 📌 Project Overview

This project provides an end-to-end real estate analytics solution that combines:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Interactive Visualizations
* Geographic Analysis
* Machine Learning Predictions
* Streamlit Dashboard Deployment

The dashboard helps users understand market trends and estimate property values based on various features.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/anjali62510-star/Real-Estate-Analytics-Dashboard.git
cd Real-Estate-Analytics-Dashboard
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run dashboard/app.py
```
